# 저장소 지침

## 프로젝트 구조와 모듈 구성

이 저장소는 한글 동물원 게임을 위한 정적 웹 앱입니다. 백엔드, 빌드 도구, 패키지 설치 과정은 없습니다.

- `index.html`은 제목, 게임, 도감, 설정 모달 화면을 정의합니다.
- `css/style.css`는 레이아웃, 애니메이션, 반응형 화면, 색상 변수를 담당합니다.
- `js/app.js`는 게임 상태, 드래그 앤 드롭, 터치 입력, 음성 출력, 진행 기록, 화면 전환을 처리합니다.
- `animals.json`은 동물 데이터 원본입니다. 항목은 `{ "name": "...", "image": "images/file.png" }` 형식을 사용합니다.
- `images/`에는 `animals.json` 항목과 대응되는 정사각형 동물 이미지를 둡니다.
- `audio/`에는 Edge TTS로 미리 생성한 동물 이름, 글자 타일용 음절, 칭찬 문구 MP3를 둡니다.
- `tools/`에는 개발용 생성 스크립트를 둡니다.

## 실행, 점검, 개발 명령

- `python3 -m http.server 8000`: 현재 폴더를 정적 서버로 엽니다. 브라우저에서 `http://localhost:8000`에 접속합니다.
- `python3 -m json.tool animals.json >/dev/null`: 동물 데이터를 수정한 뒤 JSON 문법을 검사합니다.
- `python3 tools/generate_edge_tts_audio.py --force`: Edge TTS 한국어 음성으로 동물 이름, 글자 타일용 음절, 칭찬 MP3를 다시 생성합니다.
- `git diff --check`: 커밋 전에 공백 오류를 확인합니다.

이 앱은 `animals.json`을 `fetch()`로 읽으므로 `file://`로 열지 말고 HTTP 서버를 사용하세요.

## 코드 스타일과 이름 규칙

HTML, CSS, 자바스크립트는 공백 두 칸 들여쓰기를 사용합니다. 자바스크립트는 브라우저에서 바로 실행되는 평범한 문법을 유지하고, 파일 상단의 `'use strict'`를 보존합니다.

함수와 변수는 `startQuestion`, `pickKoreanVoice`, `updateProgressLabels`처럼 낙타식 표기를 사용합니다. CSS 클래스는 `animal-card`, `tile-tray`처럼 소문자와 하이픈을 사용합니다. 이미지 파일명은 `images/polar-bear.png`처럼 소문자 하이픈 형식으로 맞추고, `animals.json`의 경로와 정확히 일치시킵니다.

## 테스트 지침

아직 자동화된 테스트 도구는 없습니다. 변경 후에는 로컬 HTTP 서버를 켜고 실제 브라우저에서 관련 흐름을 확인하세요. 제목 화면 시작, 글자 타일 배치, 터치 입력, 음성 출력, 다음/종료 버튼, 도감 단일 연습, 설정 초기화, 진행 기록 저장을 점검합니다.

데이터를 바꾼 경우 JSON 문법을 검사하고 새 이미지 경로가 모두 존재하는지 확인합니다.

## 커밋과 풀 리퀘스트 지침

커밋 메시지는 반드시 모두 한국어로 작성합니다. 최근 기록처럼 사용자에게 보이는 변화가 드러나게 짧고 구체적으로 쓰세요. 예: `초성 힌트 대기 시간을 3초에서 5초로 변경`.

커밋은 한 가지 목적에 집중합니다. 풀 리퀘스트에는 변경 요약, 확인한 브라우저나 기기, 화면 변경 스크린샷 또는 녹화, 새 이미지나 데이터 출처를 포함합니다. 관련 이슈가 있으면 연결합니다.

## 에이전트 작업 지침

명시적인 요청이 없으면 정적 앱 구조를 유지하고 새 서버나 빌드 도구를 추가하지 않습니다. 학습 흐름이나 음성 출력 동작을 바꾸면 웹 음성 API 동작이 브라우저와 기기마다 다를 수 있으므로 실제 브라우저에서 확인합니다.

Edge TTS는 온라인 서비스를 사용해 MP3를 생성합니다. 생성된 파일은 정적 앱에서 재생하지만, 상업 배포나 대규모 배포로 바뀌면 사용 조건을 먼저 다시 확인합니다.
